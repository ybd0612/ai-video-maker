# i18n 国际化系统

## 概述

wxhb 使用自研的轻量 i18n 系统，零外部依赖，支持中文（zh）和英文（en）切换。

## 文件位置

`src/i18n/index.ts`

## 架构

### 翻译字典

两个常量对象 `zh` 和 `en`，键名完全一致（`TranslationKey` 类型约束）：

```typescript
const zh = {
  "sidebar.dragToCanvas": "拖拽到画布",
  "palette.prompt": "提示词",
  // ...
} as const;

const en = {
  "sidebar.dragToCanvas": "Drag to Canvas",
  "palette.prompt": "Prompt",
  // ...
};
```

### 语言切换

语言存储在 `settingsStore.language`（"zh" | "en"），通过 localStorage 持久化。

## useT Hook

```typescript
function useT() {
  const lang = useSettingsStore((s) => s.language);
  const dict = dictionaries[lang] ?? zh;

  function t(key: TranslationKey, vars?: Record<string, string | number>): string {
    let result = dict[key] ?? key;
    if (vars) {
      for (const [k, v] of Object.entries(vars)) {
        result = result.replace(`{${k}}`, String(v));
      }
    }
    return result;
  }
  return t;
}
```

### 使用方式

```typescript
function MyComponent() {
  const t = useT();
  return <span>{t("sidebar.settings")}</span>;
}
```

### 变量插值

支持 `{变量名}` 占位符：

```typescript
t("task.deleteConfirm", { name: task.name })
// zh: "删除「任务A」？"
// en: "Delete 'Task A'?"
```

## getTranslation — 非 React 环境

```typescript
export function getTranslation(
  key: TranslationKey,
  vars?: Record<string, string | number>
): string
```

直接从 settingsStore 读取当前语言，无需在 React 组件中使用。

## 翻译键分类

| 前缀 | 用途 | 示例 |
|------|------|------|
| `sidebar.*` | 侧边栏 | `sidebar.settings` |
| `palette.*` | 节点调色板 | `palette.prompt`, `palette.prompt.desc` |
| `task.*` | 任务管理器 | `task.title`, `task.new` |
| `canvas.*` | 画布操作 | `canvas.runWorkflow` |
| `settings.*` | 设置对话框 | `settings.apiKey`, `settings.save` |
| `panel.*` | 属性面板 | `panel.label`, `panel.temperature` |
| `node.*` | 节点通用 | `node.idle`, `node.running` |
| `prompt.*` | Prompt 节点 | `prompt.placeholder` |
| `text.*` | Text 节点 | `text.model`, `text.temp` |
| `image.*` | Image 节点 | `image.model`, `image.img2img` |
| `video.*` | Video 节点 | `video.model`, `video.fps` |
| `upload.*` | Upload 节点 | `upload.dropHint`, `upload.formats` |
| `workflow.*` | 工作流引擎 | `workflow.cycleDetected` |

## 新增翻译键规范

1. 在 `zh` 对象中添加键值对
2. 在 `en` 对象中添加**相同键名**的英文翻译
3. 键名格式：`模块.语义`，使用 camelCase
4. 如需变量，使用 `{变量名}` 语法
5. TypeScript 的 `as const` + `TranslationKey` 类型会在漏翻时报编译错误
